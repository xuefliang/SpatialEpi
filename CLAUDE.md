## Overview

SpatialEpi is an R package providing methods and data for spatial epidemiology, specifically cluster detection and disease mapping. The package is published on CRAN and maintained at github.com/rudeboybert/SpatialEpi.

**Version 2.0.0**: Migrated from deprecated `sp` package to modern `sf` package.

## Architecture

### Core Algorithms (R/ directory)

The package implements three main cluster detection methods:

1. **Kulldorff spatial scan statistic** (`kulldorff.R`, `cluster_detection.cpp`)
   - Identifies spatial clusters using binomial or Poisson likelihood ratio tests
   - Monte Carlo simulation for p-value calculation
   - C++ implementation in `cluster_detection.cpp` for performance

2. **Besag-Newell method** (`besag_newell.R`, `cluster_detection.cpp`)  
   - Detects disease clusters based on k nearest cases
   - C++ `besag_newell_internal()` for core computation

3. **Bayesian cluster detection** (`bayes_cluster.R`, `bayes_cluster.cpp`)
   - Wakefield-Kim (2013) model using MCMC
   - Estimates prior/posterior probabilities of cluster membership
   - Heavy C++ implementation with RcppArmadillo

### Supporting Modules

- `zones.R` - Creates candidate zone sets from geographic areas
- `expected.R` - Computes internally standardized expected disease counts
- `create_geo_objects.R` - Prepares geographic data structures for algorithms
- `latlong2grid.R`, `grid2latlong.R` - Coordinate system conversions
- `eBayes.R`, `EBpostdens.R`, `EBpostthresh.R` - Empirical Bayes estimation
- `plotmap.R`, `mapvariable.R` - Visualization utilities

### C++ Code (src/ directory)

Performance-critical computations use Rcpp/RcppArmadillo:
- `cluster_detection.cpp` - Likelihood calculations, Monte Carlo sampling, Besag-Newell internals
- `bayes_cluster.cpp` - MCMC sampling for Bayesian model
- `RcppExports.cpp` - Auto-generated Rcpp glue code

### Data (data/ directory)

Three epidemiological datasets in `.rda` format (with `_sf` versions):
- `NYleukemia` - Leukemia cases in upstate New York census tracts
- `pennLC` - Pennsylvania lung cancer by county with demographic strata
- `scotland` - Scottish lip cancer data

## Key Dependencies

- `sf` - Modern spatial data classes (sf, sfc, sfg)
- `spdep` - Spatial dependence analysis (poly2nb, nb2mat) - works with sf objects
- `Rcpp` / `RcppArmadillo` - C++ integration for performance
- `MASS` - Negative binomial GLM (glm.nb)
